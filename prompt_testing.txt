Perintah Utama:Gunakan server MCP composio dan panggil toolkit cloudflare untuk membuat sebuah lingkungan kontainer terisolasi (Cloudflare Sandbox SDK) di awan.Instruksi Pengaturan Sandbox:Sinkronisasikan seluruh direktori kode Kotlin lokal saya saat ini ke dalam kontainer Cloudflare tersebut.Di dalam sandbox Cloudflare tersebut, pasang lingkungan pengujian: Android CLI SDK Tools dan JDK 17.Eksekusi Pengujian & Analisis Dokumen:Jalankan automated testing di dalam sandbox Cloudflare tersebut dengan menganalisis dan mengikuti 3 file panduan di workspace saya: TESTING-GUIDELINES.md, TESTING-CHECKLIST.md, dan TESTING-COVERAGE-MAP.md.Target Simulasi: Atur profil emulator kustom agar sesuai dengan spesifikasi tablet 11 inci (resolusi 1200x1920, rasio 16:10, RAM 4GB, Snapdragon 680, Android 13).Skenario Jaringan: Jalankan tes interaksi UI Jetpack Compose, lalu lakukan manipulasi jaringan (network throttling) ke mode Slow 3G dan Offline total.Kompilasi & Pengambilan File APK:Jika seluruh item di TESTING-CHECKLIST.md berstatus lulus (PASS), jalankan perintah Gradle ./gradlew assembleRelease langsung di dalam terminal sandbox Cloudflare tersebut.Setelah proses kompilasi selesai, gunakan fungsi transfer file dari Composio/Cloudflare untuk mengunduh berkas .apk final yang dihasilkan ke dalam folder lokal saya saat ini.Tampilkan hasil ringkasan pengujian serta log kesalahan (Logcat) di jendela terminal OpenCode ini jika ada bagian yang gagal (FAIL).
